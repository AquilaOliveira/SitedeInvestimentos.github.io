# 💰 Site de Investimentos

Projeto de um site informativo sobre **investimentos e educação financeira**, desenvolvido com tecnologias web básicas e publicado utilizando **GitHub Pages**.

🔗 **Acesse o site:**
https://aquilaoliveira.github.io/SitedeInvestimentos.github.io/

---

## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de apresentar conceitos básicos de **investimentos**, ajudando usuários iniciantes a entender melhor o mundo das finanças e as diferentes formas de aplicar dinheiro.

O site apresenta informações sobre tipos de investimentos, conceitos financeiros e conteúdos educativos voltados para quem deseja começar a investir.

## 🚀 Como Executar o Projeto

1. Clone este repositório:

```bash
git clone https://github.com/aquilaoliveira/SitedeInvestimentos.github.io.git
```

2. Acesse a pasta do projeto:

```bash
cd SitedeInvestimentos.github.io
```

3. Abra o arquivo **index.html** no navegador.
